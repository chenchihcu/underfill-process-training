1. Underfill 製程概論

Underfill 是應用於 BGA / Flip Chip 封裝的底部填充製程，主要目的在於分散晶片與基板間的熱應力 (Thermal Stress)，提升可靠度 (Reliability)。

製程目的

  • 分散焊接點 (Solder Joint) 應力
  • 降低衝擊/彎曲導致的失效風險 (Drop/Mechanical Shock)
  • 提升溫度循環 (TCT) 壽命
  • 防止濕氣/污染物侵入

適用範圍

BGAFlip ChipCSPPoP

涵蓋消費性電子、車用電子、網通設備等高可靠度應用。

核心觀念：Underfill 不是單純的膠水填充，而是與焊點共同承擔機械與熱應力的結構材料。

2. 材料規格 (UF3808)

UF3808 為環氧樹脂 (Epoxy) 系 Underfill，需建立完整材料特性資料庫以作為製程參數設定依據。

Tg (Glass Transition Temperature)
120–140°C
CTE (α1 / α2)
30 / 110 ppm
Modulus (儲能模數)
6–8 GPa
Curing Shrinkage
< 0.5%

材料篩選重點

  • Tg / CTE 匹配： Tg 需高於最高操作溫度；CTE 需與 solder ball / substrate 匹配，避免殘留應力 (Residual Stress) 導致 Warpage
  • Curing Shrinkage： 過高的固化收縮率會誘發 Warpage 與 Die Crack 風險
  • Flowability (流動性)： 影響 Fine Pitch / 小 Gap 的填充能力
  • Filler Content： 影響 Modulus 與 CTE 平衡
Tg 以下為玻璃態 (Glass State)，剛性高但較脆；Tg 以上為橡膠態 (Rubbery State)，CTE 顯著增加，需綜合評估可靠度需求。

4. Plasma 清洗與表面處理

Plasma Cleaning 為 Underfill 關鍵的前處理步驟，目的在清除有機汙染物並活化表面，提升流動性與附著力。

Plasma 參數

  • 氣體：Ar / O₂ / Ar+O₂ 混合
  • 功率：100–300W
  • 處理時間：60–300s
  • 壓力：0.1–1.0 Torr

適用場景

  • Fine Pitch (< 150μm) 必需
  • 小 Gap (< 50μm) 推薦
  • Flow Striation 異常時優先排查
  • 新 PCB 板廠導入時驗證
Plasma 後的 Staging Time 必須嚴格管控。超過時間未點膠，表面會重新吸附汙染物，導致 Flow Issue 與 Void。

Staging Time 管理

條件建議上限超過處理
一般環境 (Class 100K)4 小時重工 Plasma
低濕控制 (< 40%RH)8 小時重工 Plasma
充氮保存12 小時評估後決定

5. 點膠製程 (Dispensing)

點膠為製程核心，需依據產品尺寸、Pitch、Gap 決定 Pattern 與參數。

關鍵參數

參數影響
預熱溫度黏度、流動速度
針頭尺寸 (Gauge)出膠量、fillet 形狀
Valve Pressure出膠穩定性
Needle Gap (Z)膠條寬度、Racing Effect
Dispensing SpeedFillet 均勻性
Substrate Temp流變行為

品質指標 (KPI)

  • 重量 Cpk ≥ 1.33（目標 ±0.3mg）
  • Fillet 寬度符合 50%–75% 元件邊緣
  • 無 Air Trap / Void
  • 連續點膠無塞針
點膠參數的起點設定應從材料 TDS 建議值開始，再依據實際產品進行 DOE 驗證與調整。

6. 固化製程 (Curing)

固化使 Underfill 達到最終機械強度。固化 schedule 需依材料供應商建議設定。

Cure Temp
150–165°C
Cure Time
30–60 min
Ramp Rate
≤ 3°C/s
Cool Down
自然冷卻或受控

固化注意事項

  • 升溫速率過快可能導致 Thermal Shock 與 Void 生成，需控制 Ramp Rate
  • 烘箱 (Oven) 溫度均勻性需定期驗證（±3°C）
  • 若採用 In-line Curing，需確認鍊速與溫度曲線的對應關係
  • 固化後不可急冷，避免 Warpage 增加
Curing Shrinkage 與 Residual Stress 為一體兩面，製程窗口需在固化度與殘留應力間取得平衡。

8. 點膠 Pattern 與應用決策樹

依據產品設計選擇適當的點膠 Pattern：I 型、L 型、U 型。

I 型
單邊直線

L 型
雙邊轉角

U 型

決策流程

條件建議 Pattern說明
Die Size < 10mm × 10mmI 型簡單流道，單邊即可
10–20mm / Pitch 中等L 型雙邊轉角，避免 Air Lock

9. 檢驗標準 (Visual / X-Ray / CSAM)

檢驗分為三個層級，依產品可靠度要求決定檢驗項目與抽樣頻率。

Visual (外觀)

  • Fillet 連續性：100% 連續，無斷膠/缺口
  • Fillet 寬度：50%–75% 元件邊緣
  • 無膠體溢出至元件上方
  • 無明顯 Flow Striation
  • 檢驗方式：100% 顯微鏡

X-Ray/AXI

  • Void Rate：單顆 ≤ 5% 面積，整體 ≤ 3%
  • 無 Cluster Void
  • 無 Interface Void（焊點界面）
  • 檢驗方式：取樣/全檢（依客戶規範）

CSAM (超音波)

  • 適用於高可靠度/車用產品
  • 檢測 Delamination 分層
  • 檢測 Die Crack
  • 檢測 Interface Separation
  • 抽樣頻率：依客戶規範

判定標準 (Accept / Reject)

缺陷類型允收 (Accept)拒收 (Reject)
Void 面積≤ 5% 單顆，總和 ≤ 3%超過上述值或 Cluster Void
Interface Void無任何 Interface Void
Fillet 不連續無任何斷膠
Crack / Delamination無任何 Crack 或 Delam
Fillet 溢膠無溢膠至元件上方

10. 常見問題與對策

問題可能原因對策
Void / Air Trap Pattern 設計不良、Plasma 不足、Staging Time 過長、參數偏移 檢討 I/L/U 決策，確認 Plasma 參數與 Staging 時限，調整 Dispense Speed 與 Dwell Time
Flow Striation Plasma 表面能不足、材料變質、溫度不均 確認 Plasma 處理後 Water Contact Angle，檢查材料效期與儲存條件，校正加熱平台溫度
Fillet 不均勻 Needle Gap 變異、Speed 不穩、Valve 異常 校正 Z 軸高度，檢查 Valve 氣壓穩定度，確認 Dispense 路徑直線度
Warpage 過大 Curing Shrinkage × CTE mismatch、降溫過快 調整 Cure Schedule、材料 CTE 匹配，控制 Ramp Down Rate
塞針 (Needle Clog) Staging Time 過長導致材料在針頭固化 縮短停機時間，建立 Purge 頻率，檢查針頭加熱溫度
Racing Effect 黏度不足 + 壓力過大會使膠體先沿兩側流動 降低 Presure，提高基板溫度以加速固化，確認 Needle Gap
發現問題時依 “Stop → Investigate → Correct → Confirm” 四步驟排查，不可直接調整參數後連續生產。

11. 實戰案例

以下為三個典型案例，模擬從參數設定到異常排查的完整流程。

案例 A：新產品導入 (NPI)

情境： 導入 15mm × 15mm BGA，Pitch 0.4mm，Standoff 80μm

決策：

  1. 選用 L 型 Pattern（中等尺寸，標準應用）
  2. 設定基板溫度 80°C，Needle Gap 0.1mm
  3. DOE 驗證 Dispense Speed 與 Pressure，確認重量 Cpk
  4. Visual 確認 Fillet 覆蓋率 ≥ 60%
  5. X-Ray 抽樣 5pcs，Void ≤ 3%

案例 B：Void 異常排查

情境： 產線回報某批次 Void Rate 從 2% 升至 8%

排查步驟：

  1. 確認 Plasma 參數（功率/時間/氣體流量）無異常
  2. 確認 Staging Time 記錄是否在 4h 內
  3. 針對 Void 位置做 X-Ray 定位分析
  4. 發現 Void 集中於遠端角落 → 判定為 Air Trap
  5. 對策：增加前段 Dwell Time 2s，調整 Dispense Speed -10%
  6. 追蹤 3 批次驗證 Void Rate 回降至 1.5%
Void 分析時需區分“製程性 Void”（Air Trap、Flow Issue）與“材料性 Void”（Outgassing、Moisture），對策方向不同。

案例 C：Flow Striation 改善

情境： 新 PCB 板廠導入後出現 Fillet 表面條紋

排查：

  1. Water Contact Angle 量測：表面能偏低 (55°)
  2. 確認新板廠清洗製程變更，殘留汙染物增加
  3. 對策：增加 Plasma 處理時間 30%，Ar/O₂ 比例調整
  4. WCA 降至 < 20°，條紋消失
